
# Análisis de Vehículos - Aplicación Web

## Descripción del Proyecto

Esta aplicación web interactiva permite explorar y analizar un conjunto de datos de anuncios de venta de vehículos. Desarrollada con **Streamlit**, ofrece visualizaciones dinámicas y filtros interactivos para obtener insights sobre el mercado automotriz.

## Características Principales

- 📊 **Análisis Exploratorio de Datos (EDA)** completo
- 📈 **Visualizaciones interactivas** con Plotly
- 🔍 **Filtros dinámicos** por marca, modelo, año, etc.
- 🌐 **Aplicación web desplegada** en Render
- 📱 **Interfaz responsive** y fácil de usar

## Tecnologías Utilizadas

- **Python 3.x**
- **Streamlit** - Framework para aplicaciones web
- **Pandas** - Manipulación y análisis de datos
- **Plotly** - Visualizaciones interactivas
- **Git/GitHub** - Control de versiones
- **Render** - Despliegue en la nube
